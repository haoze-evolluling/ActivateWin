### **使用方法**
1. **安装密钥**  
   以管理员身份打开命令提示符，执行以下命令：  
   ```cmd
   slmgr /ipk [密钥]
   ```

2. **设置 KMS 服务器**  
   ```cmd
   slmgr /skms kms.example.com
   ```

3. **激活系统**  
   ```cmd
   slmgr /ato
   ```

4. **验证激活状态**  
   ```cmd
   slmgr /xpr
   ```
